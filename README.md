# Key to Keys

*Practice is not a chore to be completed; it is an active meditation where the physical and the spiritual collide.*

Key to Keys is a musical companion designed to move you away from rote memorization and towards a felt sense of harmonic gravity. It combines sleek, Japandi-inspired aesthetics with functional ear training and theory visualization to help you enter a flow state with your instrument.

## Features

### 🎯 Tonic Target Practice
A game of musical presence.
- **The Essence**: Stop naming notes; start feeling their function.
- **The Loop**: Establish a key, hear a target pitch, and identify its scale degree relative to the tonic.
- **The Goal**: Internalize the "gravity" of music so you can hear melody as clearly as you see color.

### 🕸️ Interactive Chord Grid
A dynamic map of harmonic possibilities.
- **Visual Theory**: Explore the diatonic chords of Major, Natural Minor, Harmonic Minor, and Melodic Minor scales.
- **Functional Relationships**: See how Roman Numerals, chord names, and modes interconnect.
- **Responsive Design**: A clean, distraction-free interface to reference while you play.

## Getting Started

This project is built with React, TypeScript, and Vite.

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Start the development server**
   ```bash
   npm run dev
   ```

3. **Build for production**
   ```bash
   npm run build
   ```
